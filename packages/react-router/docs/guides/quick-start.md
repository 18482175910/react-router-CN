## Installation 安装

React Router 核心组件已经发布到 [npm](https://npm.im/react-router) ，你可以通过 `npm` 或 [`yarn`](https://yarnpkg.com) 安装。

```sh
npm install react-router
# 或
yarn add react-router
```
