# Static Routes | 静态路由

前几个版本的React Router使用静态路由去配置你的应用的路由。这允许在渲染之前对路由进行检查和匹配。自从v4版本用动态组件取缔路由配置，一些以前的用例现在变得不太明显而且比较棘手。

我们正在开发一个使用静态路由配置和React Router的包，以继续满足这些用例。这个包现在正在开发中，但我们很乐意为您尝试并帮助您。
